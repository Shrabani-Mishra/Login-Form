# Responsive Login Form

This is a **responsive login form** built using **HTML**, **CSS**, and **JavaScript**.  
The form features a clean and modern user interface designed to work seamlessly on desktop and mobile screens.

## 🔧 Features
- ✔️ Responsive layout
- ✔️ Simple and intuitive UI
- ✔️ Clean form design
- ✔️ CSS animations and hover effects
- ✔️ Easy to customize styles

## 💻 Technologies Used
- HTML
- CSS (Flexbox & Media Queries)
- JavaScript (Form validation logic can be added)

🔗 https://shrabani-mishra.github.io/Login-Form/

## 🚀 How to Use
1. Clone the repository  
2. Open `index.html` in your browser  
3. Customize the form styles as needed
